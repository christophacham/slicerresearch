# Frameworks Used in BambuStudio

## Core Frameworks
- **wxWidgets**: GUI framework for cross-platform desktop application development
- **OpenGL**: Graphics rendering for 3D visualization and UI components
- **GLFW3**: Window management and OpenGL context creation
- **GLEW**: OpenGL Extension Wrangler for accessing OpenGL extensions

## Build and Development Frameworks
- **CMake**: Cross-platform build system
- **Ninja**: Build system generator (alternative to Make)
- **Xcode**: Build system for macOS (when using Xcode generator)
- **Visual Studio**: Build system for Windows (when using MSVC)

## Graphics and Visualization
- **OpenVDB**: Volumetric data processing library
- **Eigen3**: C++ template library for linear algebra
- **libigl**: Geometry processing library
- **OpenCASCADE Technology (OCCT)**: 3D modeling and CAD operations

## Networking and Communication
- **CURL**: HTTP/HTTPS client functionality
- **OpenSSL**: SSL/TLS encryption support
- **Boost.Beast**: Asynchronous networking library

## Data Processing and Utilities
- **Boost**: C++ libraries for system programming
- **Intel TBB**: Threading Building Blocks for parallel computing
- **NLopt**: Nonlinear optimization library
- **Cereal**: C++11 serialization library
- **ZLIB**: Compression library
- **PNG**: Image format support
- **FFmpeg**: Video/audio encoding/decoding (avcodec, avutil, swscale, swresample)
