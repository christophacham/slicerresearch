# Languages Used in BambuStudio

## Primary Languages
- **C++**: Core application logic and performance-critical components
- **CMake**: Build system configuration
- **Perl**: Used for build dependencies and unit/integration tests (via XS modules)
- **Shell Scripts**: Build scripts for Linux (BuildLinux.sh) and Mac (BuildMac.sh)
- **Batch Scripts**: Windows build script (build_win.bat)
- **Dockerfile**: Containerization

## Supporting Technologies
- **C++17**: Modern C++ standard used throughout the codebase
- **Perl**: Used for build dependencies and testing infrastructure
- **Shell/Batch**: Platform-specific build and deployment scripts
